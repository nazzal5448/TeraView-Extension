# TeraView - TeraBox Link Preview (Edge)

A lightweight utility for previewing public TeraBox share links before opening them.

## Installation (Unpacked)
1. Open Edge and navigate to `edge://extensions/`
2. Enable "Developer mode" in the bottom left corner.
3. Click "Load unpacked" and select this `edge-extension` directory.
4. Pin the extension to your toolbar.

## Features
- Validates TeraBox URLs
- Fetches and displays OpenGraph metadata (title and thumbnail)
- Auto-focus and clipboard reading for seamless experience
- Clean, minimal, modern UI
- Strictly follows Manifest V3 guidelines
